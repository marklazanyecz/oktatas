Egy css blokk felépítése

szelektor {
    tulajdonság: érték;
}

A szelektorok ugyan azok mint a formázandó állományban a tagek.

Minden szelektornak meghatározott tulajdonságaik vannak, 
amelyekhez korlátozottak a használható értékek.

Alap szelektorokra példák:
h1 { kód }
#valamiid { kód }	Azonosítóval ellátott elem, html-ben id="" paraméterrel
.osztaly { kód }    Osztályba sorolt elem, html-ben class="" paraméterrel


BETŰFORMÁZÁS
p {
  font-family: "Arial", "Helvetica", sans-serif;
  font-size: 16px;              px, em, rem, %
  font-weight: bold;            normal, bold, bolder, lighter, 100-900
  font-style: italic;           normal, italic, oblique
  font-variant: small-caps;     small-caps, normal
  line-height: 1.5;             Sortávolság
}
Lehetséges mértékegységek:
px - pixelméret
em - szülő elem betűméretének aránya
rem - a html dokumentum betűméretének aránya
% - százalékos arány
